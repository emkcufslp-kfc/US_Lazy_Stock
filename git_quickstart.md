# Git Quickstart

## Create local repo
```bash
git init
git add .
git commit -m "Initial US stock monitor project"
git branch -M main
```

## Connect to GitHub
```bash
git remote add origin <your-github-repo-url>
git push -u origin main
```

## Then open in Codex
1. Connect GitHub in ChatGPT / Codex
2. Select this repository
3. Continue development there
